mastodon-archive-search

a basic web ui for searching your own posts from an exported archive

how to use

  1. request + download your archive (reference)
  2. extract the archive (if you're not sure how to extract .tar.gz files, try using 7zip or other options listed here)
  3. add the extracted file outbox.json to the "archive" input below
  4. type a query in the "search" input below

notes

  • this app does not collect or store any data remotely
  • your most recently added outbox is saved to your browser's local storage, so you only need to request + add a new archive when you want to see new posts
  • media previews are shown only as alt text
  • fields searched include:
    • the post text
    • the post content warning
    • the alt text of any attached media

  • Search results will be displayed here
